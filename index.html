    How to use the To-do List

    1. Add tasks using the input field.
    2. Press "Enter" or click "Add Task" to add a task.
    3. As you work on your tasks, you can mark them as completed. This helps you keep track of your progress. If you no longer need a task, you also have the option to delete it.

    That's it! You're now ready to enjoy using the To-Do List to stay organized and get things done.
